# 飞机大战 - doubao-version

经典飞机大战游戏，使用 HTML5 Canvas 实现，无需任何依赖，浏览器直接打开即玩。

## 游戏简介

控制你的战机消灭不断涌来的敌机，收集道具增强火力，挑战更高的分数！

## 功能特性

- **三种敌机类型**: 基础型（红色三角）、中型（橙色方块，会射击）、重型（紫色圆形，血厚会射击）
- **Boss 战**: 每隔几个关卡出现 Boss，发射弹幕攻击
- **道具系统**: 火力增强(P)、生命恢复(+)、护盾(S)
- **三级火力**: 从单发到五发散射
- **关卡系统**: 每 2000 分升一级，敌人变强变多
- **最高分记录**: localStorage 持久化保存
- **暂停功能**: P 键或 Esc 键暂停/继续
- **多端支持**: 键盘、鼠标、触屏均可操作

## 技术栈

- HTML5 Canvas
- 原生 JavaScript（无框架依赖）
- CSS3

## 如何运行

### 方式一：直接打开
双击 `index.html` 文件，用浏览器打开即可。

### 方式二：本地服务器
```bash
cd plane-war-doubao-version
python3 -m http.server 8091
# 浏览器访问 http://localhost:8091
```

## 操作说明

| 操作 | 键盘 | 鼠标/触屏 |
|------|------|-----------|
| 移动 | WASD / 方向键 | 移动鼠标/手指滑动 |
| 射击 | 自动射击 | 自动射击 |
| 暂停 | P / Esc | - |

## 游戏机制

- 初始 3 条命，受伤后短暂无敌闪烁
- 击杀敌机有几率掉落道具
- 火力道具最多叠加到 3 级
- 每 2000 分升一级，敌人生成更快更强
- Boss 在关卡 3 之后每 3 关出现一次
